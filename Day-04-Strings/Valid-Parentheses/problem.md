#  Valid Parentheses

## Problem Statement
Given a string containing just the characters:
'(', ')', '{', '}', '[' and ']'

Determine if the input string is valid.

A string is valid if:
- Open brackets are closed by the same type
- Open brackets are closed in the correct order

---

## Example 1
Input: "()"
Output: true

## Example 2
Input: "()[]{}"
Output: true

## Example 3
Input: "(]"
Output: false
