# 25. Reverse Nodes in k-Group

## Problem Statement
Given the head of a linked list, reverse the nodes of the list k at a time, and return the modified list.

Only nodes in full groups should be reversed. If the number of nodes is not a multiple of k, leave the remaining nodes as is.

---

## Example
Input:
1 -> 2 -> 3 -> 4 -> 5
k = 2

Output:
2 -> 1 -> 4 -> 3 -> 5
