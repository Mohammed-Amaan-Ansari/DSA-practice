#  Container With Most Water

## Problem
Find two lines that together with the x-axis forms a container,
such that the container contains the most water.

## Example
Input:
[1,8,6,2,5,4,8,3,7]

Output:
49

## Approach
1. Use two pointers.
2. Calculate area.
3. Move smaller height pointer.

## Complexity
- Time: O(n)
- Space: O(1)
